浏览器禁止本地脚本

你的浏览器拒绝在 file:// 下加载本图谱所需的 JS 资源。

请改用 Chrome 或 Firefox 打开本文件，或在 wiki/ 目录下跑：

python3 -m http.server 8000

再访问 http://localhost:8000/knowledge-graph.html。

建议桌面浏览器打开

图谱在桌面浏览器效果最佳。当前屏幕过窄（< 768px），交互将受限。
